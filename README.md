# JobFit — AI-Powered Job Hunting Platform

Match your resume to real jobs, identify skill gaps, and 
track your entire application pipeline in one place.

**Live demo:** [jobfit.vercel.app](https://jobfit.vercel.app)

## Features
- AI match scoring — compare your resume against any job description
- Real job search — live listings from LinkedIn, Indeed, Glassdoor  
- Skills gap tracker — see which skills appear missing most often
- Application tracker — Kanban board for your whole pipeline
- Resume quality scoring — AI grades your resume before you apply

## Tech Stack
React · Node.js · Express · PostgreSQL · Prisma · 
Redis · Docker · Nginx · GitHub Actions · Gemini AI

## Running locally
\`\`\`bash
# Clone
git clone https://github.com/YOURUSERNAME/jobfit.git

# Install frontend
cd client && npm install && npm run dev

# Install backend  
cd server && npm install && npm run dev
\`\`\`
